# PhishGuard AI 🛡️

A web application that detects phishing attempts across URLs, links, emails, and messages — helping users identify malicious content before they fall victim to it. Built as part of the Infosys Springboard internship program.

## Features

✨ Key Features:

🔗 Detects phishing URLs and links
📧 Detects phishing emails
💬 Detects phishing messages
⚡ Fast, modern frontend built with Vite
🎨 Clean UI styled with Tailwind CSS
🚀 Deployed live on Netlify

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS

## Getting Started

### Prerequisites

- Node.js installed

### Quick Setup

```bash
git clone https://github.com/thanniruvaishnavi/PishGuard-AI.git
cd PishGuard-AI
npm install
```

### Running the App

```bash
npm run dev
```
